# 🛒ONLINE STORE

<p>This is a React (Frontend) and Node.js (Backend) application using Redis and MongoDB, deployed with Docker Swarm to provide load balancing and high availability.</p>

## Prerequisites

<li>Java JDK 8 or higher</li>
<li>MySQL Server</li>
<li>IntelliJ IDEA</li>
<li>Docker</li>
